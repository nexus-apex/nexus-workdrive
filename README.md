# NexusWorkDrive

Cloud File Storage - Better than Google Drive

Part of [Nexus Suite](https://nexussuite.ogts.in) - 75 Enterprise SaaS Apps.

## Tech Stack
- **Backend**: Django 5.x + DRF + PostgreSQL 17 + pgvector + Redis 7 + Celery 5
- **Frontend**: Next.js 15 + React 19 + TypeScript + Tailwind v4 + shadcn/ui
- **Mobile**: Expo SDK 52 + React Native 0.76
- **Infra**: Docker + Coolify + GitHub Actions + Traefik + Cloudflare + R2

## Deploy
Auto-deployed via GitHub Actions to Coolify webhook on every push to `main`.
